# Code Examples

This directory contains working HTML and JavaScript examples for the SelfLearning course.

## HTML Examples

### [LinksImagesAndTitle.html](LinksImagesAndTitle.html)
Demonstrates basic HTML structure with:
- Page title
- Links to external sites
- Image embedding (both online and local images)

## JavaScript Interaction Examples

### [button-click.html](button-click.html)
Shows basic button click interactions:
- Changing text content dynamically
- Modifying element styles with JavaScript
- Event listeners and handlers
- Reset functionality

**Concepts:** Event handling, DOM text manipulation, style changes

### [form-handling.html](form-handling.html)
Demonstrates form data handling:
- Collecting input from text fields
- Preventing default form submission
- Displaying form data dynamically
- Form validation

**Concepts:** Form events, input values, preventing defaults

### [dom-manipulation.html](dom-manipulation.html)
Advanced DOM manipulation techniques:
- Creating new elements dynamically
- Adding/removing elements from the page
- Toggling visibility
- Changing multiple properties
- Resetting state

**Concepts:** DOM creation, element manipulation, classList, appendChild, removeChild

## How to Use These Examples

1. **View in Browser**: Open any `.html` file directly in your web browser
2. **View Source**: Right-click and select "View Page Source" to see the code
3. **Experiment**: Try modifying the code and refresh to see changes
4. **Learn**: Each example builds on concepts from the course lessons
