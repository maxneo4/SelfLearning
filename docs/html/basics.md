## What is HTML?

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It describes the structure of a web page using a series of elements called tags.

## HTML Document Structure

Every HTML document follows a basic structure:

```html
<html>
<head>
    <title>Page Title</title>
</head>
<body>
    <!-- Your content goes here -->
</body>
</html>
```

### Key Components:

- `<html>` - The root element that wraps all content
- `<head>` - Contains metadata and document information
- `<title>` - Sets the page title (shown in browser tab)
- `<body>` - Contains the visible page content

## Common HTML Tags

### Headings

HTML provides six levels of headings:

```html
<h1>Main Heading</h1>
<h2>Subheading</h2>
<h3>Sub-subheading</h3>
<!-- h4, h5, h6 also available -->
```

### Paragraphs

Use `<p>` tags for paragraphs of text:

```html
<p>This is a paragraph of text.</p>
```

### Line Breaks

Add a line break with `<br>`:

```html
<p>First line<br>Second line</p>
```

### Comments

Add notes in your code that won't be displayed:

```html
<!-- This is a comment -->
```
